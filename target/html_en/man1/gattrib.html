NAME

gattrib - gEDA/gaf 原理图属性编辑器

语法

gattrib[选项...][文件...]

描述

gattrib 是一个类似电子表格的批量属性编辑器，通常用于修改在gschem(1) 中创建的属性。

如果在命令行上没有指定要打开的文件s ，则 gattrib 会在启动时显示文件选择器对话框。

选项

-q, --quiet

安静模式。关闭所有警告/注意/消息。

-v, --verbose

详细模式。输出所有的诊断信息。

-h, --help

打印帮助信息。

环境变量

GEDADATA

指定 Scheme 和 rc 文件的搜索目录。默认是 ‘${prefix}/share/gEDA’。

GEDADATARC

指定 Scheme 和 rc 文件的搜索目录。默认是‘$GEDADATA’.
